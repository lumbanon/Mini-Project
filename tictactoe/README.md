# Tic Tac Toe (C++)

This repository contains a simple Tic Tac Toe game written in C++.

## Description
This is a console-based Tic Tac Toe game where two players take turns placing X and O on a 3x3 board. The program checks for winning conditions and detects when the game ends in a draw.

## Features
- Two-player gameplay
- 3x3 board displayed in the terminal
- Win condition checking
- Draw detection

## How to Run

1. Compile the program:

g++ tictactoe.cpp -o tictactoe

2. Run the program:

./tictactoe
